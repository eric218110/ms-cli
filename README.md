# MS-CLI

MS-CLI is nodeJS Cli for create micro services.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) or [NPM](https://docs.npmjs.com/cli/v6/commands/npm-install) to install MS-CLI.

## Usage

```bash
$ ms create micro-service-name
```
